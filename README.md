# Voided Dimension

Un mod de Minecraft Fabric que añade una nueva dimensión del vacío con características únicas.

## Características

- Nueva dimensión personalizada con generación de mundo única
- Bloques y items personalizados
- Entidad Puppetman con animaciones GeckoLib
- Sistema de alimentación y efectos personalizados

## Requisitos

- Minecraft 1.21.1
- Fabric Loader
- Fabric API

## Instalación

1. Descarga el mod desde las releases
2. Coloca el archivo JAR en tu carpeta de mods
3. Inicia Minecraft con el perfil de Fabric

## Desarrollo

Este mod está desarrollado usando:
- Fabric Mod Loader
- GeckoLib para animaciones de entidades
- Sistema de datagen para generación de recursos
